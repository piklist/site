Piklist 0.9.3.2 released

Home » News » Piklist 0.9.3.2 released

Apr 25

Piklist version 0.9.3.2 is now available at WordPress.org, and addresses a few issues. When installed with BuddyPress, Piklist would sometimes stop user registration emails from going out, which of course is a pretty big issue for your users. The Piklist update feature did not support running updates for brand new plugins. We plan on releasing a plugin in the next few weeks which will require this feature so we wanted to get this fix out as soon as possible. Version 0.9.3.2 also fixed an annoying notice that popped up when clicking on a quick link in a list table.

Additionally, we updated the built in demos to take advantage of WordPress 3.9’s drag-and-drop image uploader for the Editor field.

Enjoy!
