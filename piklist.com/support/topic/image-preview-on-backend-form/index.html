      Hey @eflouret!

      I believe how you’re changing the preview size should work… Just to be clear, newsleter is a registered image size?

      For the basic uploader, no there is no preview. That’s just using the standard HTML file input field, which doesn’t offer that. Hence, basic. ðŸ™‚

      Yes! You can do that using the limit sanitization:

      
array(
 'type' => 'file'
 ,'scope' => 'post_meta'
 ,'field' => 'article_image'
 ,'label' => 'Article Image'
 ,'options' => array(
	'modal_title' => 'Add File(s)'
	,'preview_size' => 'newsletter'
 ),
 ,'sanitize' => array(
  array(
   'type' => 'limit'
   ,'options' => array( 'max' => 1 )
  ) 
 )
),

      Hope this helps!
