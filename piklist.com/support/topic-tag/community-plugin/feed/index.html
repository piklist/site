]]> Select2 (http://ivaynberg.github.com/select2/) is a great jQuery plugin for advanced single-select, multi-select and tag-type select boxes.

I have a very simple working example by simply updating ‘class-piklist-form.php’ to alias ‘select2’ as ‘select’ and to enqueue ‘select2.min.js’ and ‘select2.css’. Last change was updating ‘pik.js’ to call the jQuery $.select2() method for fields of type ‘select2’ just like you do for datepicker, timepicker, etc.

I can publish my ‘fork’ of the code to GitHub if you’d like, but at the very least thought I would send this in as a suggestion as it will probably solve many ‘selection’ use cases.

Cheers,

Adam Anderly

]]> Hello guys,
besides the ‘switch’ field type, i also created the ‘slider’ type.
It can produce a nice slider to set a value between a range

download (both) here: piklist-slider-and-switch.zip

Install as normal WP plugin
Customize looks via the css
Use as :

piklist('field', array(
'type' => 'slider'
,'field' => 'ranger'
,'label' => 'Range '
,'description' => 'Set width of the border'
,'options' => array(
'min_value' => '100',
'max_value' => '1000',
'step_value' => '100'
)
));

Any comments or ideas welcome
Enjoy!

]]> Goodday,

i want to share a new PIKLIST field type with you guys.
It is called ‘switch’ and can be used for quickly turning on / off your settings via a nice ‘iphone-switch’ style.

You can get it here: Download Zip

Install it as a WP plugin and use it as:

piklist(‘field’, array(
‘type’ => ‘switch’
,’field’ => ”
,’label’ => ”
,’description’ => ”
,’attributes’ => array(
‘class’ => ‘text’
)
));
