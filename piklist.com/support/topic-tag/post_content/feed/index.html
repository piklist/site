{

]]> Hi,

I found it causes incompatibility issues with probably a large number of plugins if fields are being saved to post_content or post_excerpt. For the most part it is an inability to delete content from this fields once it has been added and saved. Experimenting a little bit, I was sometimes able to delete parts the text but I couldn’t delete it up to the last letter.

The plugins I could identify causing this were:

– Custom Field Suite (which I still use for relationship fields)
– Content Aware Sidebars
– Toolbar Publish Button

Probably others too, but I wasn’t quite sure since the exact effects are different from one to the other.

I’ve no idea what causes the problem. It’s not a huge thing to save to post meta instead, but of course it comes with a few downsides. I thought I should let you know.

Thanks – Stefan
