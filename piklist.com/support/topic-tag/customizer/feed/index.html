WordPress.org has mandated there be NO settings API based option panels of any kind for themes hosted on WordPress.org. Existing themes still using option panels will be required to remove said panels and add customizer ONLY options before updating. New themes must adhere to this new guideline, or be rejected. The WordPress.org theme review team has essentially stopped WordPress.org theme devs from using most option frameworks.
