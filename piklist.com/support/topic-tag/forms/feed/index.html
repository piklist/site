]]> Hi,

Is possible that when the ‘field_1’ changes this triggers the conditions of ‘field_2’? Actually this trigger the conditions just on unfocus and “type => update” don’t works.


piklist( 'field', [
	'type' => 'text',
	'scope' => 'post_meta',
	'field' => 'field_1',
	'label' => 'Field 1'
]);

piklist( 'field', [
	'type' => 'text',
	'scope' => 'post_meta',
	'field' => 'field_2',
	'label' => 'Field 2',

	'conditions' => [
		[
			'field' => 'field_1',
			'value' => '',
			'compare' => '!=',
			//'type' => 'update',
		]
	]
]);
