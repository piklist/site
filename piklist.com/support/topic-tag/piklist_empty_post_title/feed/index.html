]]> Hi,

The example in this tutorial works well for one single plugin.

I have two plugins that both use piklist_empty_post_title to set Title. The example code seems not work under this scenario- lower priority one will turn the Title back to empty/Auto Draft.

So I changed the last return statement, from post_arry to data, and it works. Not sure if this the right direction but FYI.

The code will be:

function set_post_title_to_order_number($data, $post_array)
{
  if ($post_array['post_type'] == 'MY-POST-TYPE')
  { 
    return 'Order #' . $post_array['ID'];
  }
  else
  {
    // return $post_array['post_title'];
    // original parameter $data contains the title passed to this function, 
    // even being modified by another plugin
    return $data;
  }
 
}
// Change priority so that two plugins use different numbers
add_filter('piklist_empty_post_title', 'set_post_title_to_order_number', 10, 2);
