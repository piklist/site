]]> When I install piklist in my wordpress installations using the language pt-BR I can’t save posts.

step-by-step:
– Install a fresh wordpress
– Change the language to “Português do Brasil”
– create a new post/page, give it a title and save. The post status will save as “draft” (here the issue).

Note¹: (Using pt-BR) I can’t change “manually” post status from Rascunho (draft) to Publicado (published).
Note²: When I change to english, the posts save as publish (default/correct behavior).

  add_filter('piklist_admin_pages', 'piklist_theme_setting_pages');
  function piklist_theme_setting_pages($pages)
  {
     $pages[] = array(
      'page_title' => __('Cabinet Door Shop')
      ,'menu_title' => __('Cabinet Door Shop', 'CDS')
      ,'capability' => 'manage_options'
      ,'menu_slug' => 'custom_settings'

      ,'menu_icon' => plugins_url('piklist/parts/img/piklist-icon.png')
      ,'page_icon' => plugins_url('piklist/parts/img/piklist-page-icon-32.png')
      ,'single_line' => true
      ,'default_tab' => 'Materials'
      ,'save_text' => 'Save Door Settings', 'Save CDS'
  );
 
    return $pages;
  }
