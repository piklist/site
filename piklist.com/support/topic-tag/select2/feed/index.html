]]> Select2 (http://ivaynberg.github.com/select2/) is a great jQuery plugin for advanced single-select, multi-select and tag-type select boxes.

I have a very simple working example by simply updating ‘class-piklist-form.php’ to alias ‘select2’ as ‘select’ and to enqueue ‘select2.min.js’ and ‘select2.css’. Last change was updating ‘pik.js’ to call the jQuery $.select2() method for fields of type ‘select2’ just like you do for datepicker, timepicker, etc.

I can publish my ‘fork’ of the code to GitHub if you’d like, but at the very least thought I would send this in as a suggestion as it will probably solve many ‘selection’ use cases.

Cheers,

Adam Anderly
