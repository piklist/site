    ðŸ™‚ Thanks Steve. That was the final piece. I am using piklist v.0.9.9.7, so piklist_part_process_callback worked. Also, looking at $part_data, I had to change the conditional to match the correct entry of the array. For anyone that might need this, my final code is below. I have two different sets of workflow tabs, one to display only on my homepage (post_id: 5) and the other set to display only on my about page (post_id: 34).

    function my_disable_metabox($part_data, $folder)
{
  

	// get post ID
	if ( isset( $_REQUEST['post'] ) ) {
		$post_id = $_REQUEST['post'];
	} elseif ( isset( $_REQUEST['post_ID'] ) ) {
		$post_id = $_REQUEST['post_ID'];
	} 
	

	// Display the correct Workflow on the correct page
	if ( ($post_id != 5) && $part_data['data']['flow'][0] == 'homepage_workflow' )  {
		return;
	} else if ( ($post_id != 34) && $part_data['data']['flow'][0] == 'about_workflow' )  {
		return;
	}

  return $part_data;
  
}
add_filter('piklist_part_process_callback','my_disable_metabox', 10, 2);

    Thanks Again!
